# Linux Server Hardening

This repository documents my hands-on work on securing a Linux server
using RHCSA-level administration and security best practices.

## Topics covered
- User and group management
- SSH hardening
- Firewall configuration (firewalld)
- SELinux basics
- System auditing concepts

## Goal
To demonstrate practical Linux administration and security-focused
configuration suitable for junior DevOps roles.

> This project is a work in progress and will be updated regularly.
